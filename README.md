# Care Link Frontend

This is the frontend project for Care Link.

## Getting Started

1. Install dependencies:
   ```
   npm install
   ```

2. Start the development server:
   ```
   npm start
   ```

## Project Structure

- `src/` - Source code for the frontend application
- `public/` - Static assets

## Technologies Used

- React
- JavaScript
- CSS
